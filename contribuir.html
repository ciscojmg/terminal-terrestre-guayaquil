Cómo contribuir

Gracias por tu interés en mejorar la información del Terminal Terrestre de Guayaquil. Este proyecto se alimenta de la comunidad a través de pull requests.

add_business Cómo agregar una cooperativa

  1. Edita el archivo data/cooperativas.json
  2. Añade un nuevo objeto siguiendo la estructura existente (ver ejemplo en el repositorio)
  3. Imágenes: Si usas imagen local, súbela en assets/images/
  4. Sitemap: Si agregas una cooperativa nueva, añade su URL en sitemap.xml

person Cómo aparecer como contribuidor

La lista de contribuidores se actualiza automáticamente desde la API de GitHub cuando se hace merge de un pull request.

No necesitas editar nada manualmente. Cuando tu PR sea aceptado, tu avatar aparecerá en la sección de contribuidores.

route Destinos populares

Los destinos se extraen automáticamente de las cooperativas. Cada destino único en rutaPrincipal.destino se muestra. No es necesario editar archivos de destinos por separado.

checklist Validación

Antes de enviar tu PR, verifica que:

  • El JSON es válido
  • No hay cooperativas duplicadas (mismo id)
  • Las URLs de imágenes son accesibles
